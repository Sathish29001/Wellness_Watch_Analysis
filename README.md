# Wellness_Watch_Analysis

📊 New Project: Wellness Data Analytics using Python 🏃‍♂️💡
I recently completed a comprehensive data analysis project on health and wellness tracker data using Python in Jupyter Notebook. By analyzing daily user metrics like steps, distance, active minutes, and calorie burn, I uncovered actionable patterns in user habits and activity levels.

🔍 Key Steps & Insights:Data Cleaning & Feature Engineering: Processed a dataset of nearly 940 records , standardized data types , and engineered time-based variables (like day of the week)  to track behavioral trends.  

Activity Categorization: Developed custom logic using numpy.select to categorize users into Sedentary, Active, and Very Active tiers based on their historical averages.  

Behavioral Observations: Found that users are highly sedentary on average (81.3% of their tracked time) , but step counts consistently peak mid-week (Wednesdays) and on Saturdays. 

Advanced Data Visualization: Built multi-variable scatter plots and pie charts using Matplotlib and Seaborn to map direct correlations between varying tiers of active minutes and overall calorie expenditure.  This project highlights how consumer wearable data can be transformed into clear, visual trends to guide data-driven wellness strategies!

🛠️ Tech Stack Used: Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook, Kaggle.
